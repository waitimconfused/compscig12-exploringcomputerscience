Exploring Computer Science (HTML Review)

Outline

Individually research up to 3 collaborative research areas using a recognized industry publication. Investigate a topic in theoretical computer science. Make webpages to report the information. Use Git / GitHub to manage project. CSS is not necessary, but HTML formatting (titles, headings, paragraphs) are needed.

Collaborative Research Areas

You must use the Association of Computing Machinists (ACM) Digital Library (search for free articles but may need to register with a free account using FAKE personally identifiable information) or the Institute of Electrical and Electronics Engineers (IEEE) Xplore digital library (filtering for Open Access Only articles) for your collaborative research areas.

Hint: Use AI to help you summarize the articles so you can see if it has the information you need.

Details

For each collaborative research area, explain

  • How the collaborative research area uses computer science
  • A positive impact of the collaborative research area on a society/culture around the world
  • A negative impact of the collaborative research area on a society/culture around the world,

Theoretical Computer Science

Theoretical computer science studies mathematical and abstract areas of computer science.

Here is list of topics https://en.wikipedia.org/wiki/Theoretical_computer_science

Details

Write 400 words explaining what the theoretical computer science topic is about. Give examples of how the topic has been applied to specific software. Create a webpage to present your information. Use multimedia elements like images, embedded videos, and links to sources.

Theoretical computer science topics include (but are not limited to):

  • Algorithms
  • Data structures
  • Computational complexity
  • Parallel and distributed computation
  • Probabilistic computation
  • Quantum computation
  • Automata theory
  • Information theory
  • Cryptography
  • Program semantics and verification
  • Algorithmic game theory
  • Machine learning
  • Computational biology
  • Computational economics
  • Computational geometry
  • Computational number theory and algebr

Managing This Project

This is an individual project.

You must use Git / GitHub to manage your code. Make sure to commit your work at numerous logical times including:

  • Creating the original files
  • Completion of each of your collaborative research areas
  • Completion of your theoretical computer science topic
